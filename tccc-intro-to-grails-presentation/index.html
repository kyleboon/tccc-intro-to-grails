What is Grails?

Grails is an Open Source, full stack, web application framework for the JVM. It takes advantage of the Groovy programming language and convention over configuration to provide a productive and stream-lined development experience.

Groovy


def withNewTransaction(Closure closure) {
    def transaction = sessionFactory.currentSession.beginTransaction()
    try {
        def result = closure.call()
        transaction.commit()
    } catch (exception) {
        log.error 'Rolling back.', exception
        transaction.rollback()
        throw all
    }

    result
}

Domain Class


class Account {
    Customer owner
    AccountType type
    LocalDate openDate
    BigDecimal balance

    Date dateCreated
    Date lastUpdated

    static constraints = {
        owner(nullable: false)
        type(nullable: false)
        openDate(nullable: false)
        balance(nullable: false)
    }
}
